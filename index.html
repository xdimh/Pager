Pager

Pager is a jQuery pagination plugin,you can easily use in your project with jQuery.

Download this project as a .zip file Download this project as a tar.gz file

How to use it

1. Add below in your html file.
<link rel="stylesheet" href="YourPath/pager_blue.css"/>
<script src="http://cdn.staticfile.org/jquery/2.1.1-rc2/jquery.min.js"></script>
<script src="YourPath/Pager.js"></script>
2. Add pager container
<div id="J-pager"></div>
3. init pager like below
<script>
  $('#J-pager').pager({
     curno : 1,
     total : 20
  });
</script>
4. result

custom pager lang

You can pass the corresponding options to custom pager lang.

 $('.J-pagerwraper2').pager({
        curno : 1,
        total:22,
        onchange:function(_data) {
            alert(_data.index);
        },
        lang: {
            firstPageText: 'Home',
            firstPageTipText: 'Home',
            lastPageText: 'End',
            lastPageTipText: 'End',
            prePageText: '<Prev',
            prePageTipText: 'Prev',
            nextPageText: 'Next>',
            nextPageTipText: 'Next',
            totalPageBeforeText: 'total',
            totalPageAfterText: 'pages',
            currPageBeforeText: 'current:',
            currPageAfterText: '',
            totalInfoSplitStr: '/',
            gopageBeforeText: '&nbsp;JumpTo',
            gopageButtonOkText: 'Go',
            gopageAfterText: ''
        }
    });

Result

You can achieve two pagers move together by using 'ref' option.

<script>
    $('.J-pagerwraper1').pager();
    $('.J-pagerwraper2').pager({
        ref : $('.J-pagerwraper1').data('xdimh.pager')
    });
</script>
